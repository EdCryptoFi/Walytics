# Walytics — Demo Video Script (2-3 min)

## Intro (0:00-0:15)
"Hi, I'm ED, and this is Walytics — a real-time analytics dashboard for Walrus decentralized storage on Sui, powered by Tatum."

## Dashboard Overview (0:15-0:45)
"Here's the main dashboard. We have four key metrics: total blobs stored, storage used, unique publishers, and average blob size. Below, the chart shows blob publishing activity over time, and the size distribution shows what kind of data is being stored."

## Tatum Integration (0:45-1:10)
"You can see the green Tatum connection status at the top — we're connected to Sui Mainnet via Tatum's RPC endpoint at sui-mainnet.gateway.tatum.io. All on-chain data flows through Tatum's enterprise-grade infrastructure."

## Blob Explorer (1:10-1:30)
"The Blob Explorer lets you browse individual blobs — publisher, size, storage type, and timestamp. You can filter and search through the data."

## AI Analytics (1:30-2:00)
"Here's the AI analytics chat. I can ask questions like 'What's the average blob size?' or 'Who are the top publishers?' and Gemini 2.0 Flash responds using the live metrics. I can also generate a full weekly analytics report with one click."

## Walrus Write-Back (2:00-2:20)
"One of the coolest features: analytics snapshots are stored back on Walrus as blobs. This creates an immutable, verifiable history of metrics. Click 'Store Snapshot' and the data goes to Walrus."

## Closing (2:20-2:45)
"Walytics is built with Next.js, Tatum Sui RPC, Walrus storage, and Google Gemini AI. Check the GitHub repo for the full code. Thanks to Tatum and Walrus for the hackathon!"
